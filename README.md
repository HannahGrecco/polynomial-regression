#  Salary Prediction with Polynomial Regression

> 🇧🇷 Prefer Portuguese? Read the **[Portuguese version](README.pt-BR.md)**.

A Machine Learning project developed during the Rocketseat ML course.

The model predicts an employee's salary based on:

- `tempo_na_empresa` (Employee tenure)
- `nivel_na_empresa` (Employee level)

## Project Workflow

-  Exploratory Data Analysis (EDA)
-  K-Fold Cross Validation
-  Linear Regression (Test)
-  Polynomial Regression
-  Scikit-Learn Pipeline
-  FastAPI REST API
-  Streamlit frontend

##  Technologies

- Python
- Pandas
- NumPy
- Scikit-Learn
- FastAPI
- Streamlit
- Joblib

##  What I Learned

Throughout this project, I learned how to:

- Apply **K-Fold Cross Validation** to evaluate Machine Learning models more reliably.
- Build and deploy a **REST API** using **FastAPI** to serve model predictions.
- Create a simple interactive web interface with **Streamlit** to consume the API and make predictions.
