#  Predição de Salário com Regressão Polinomial

> 🇺🇸 Read the **[English version](README.md)**.

Projeto desenvolvido durante o curso de Machine Learning da Rocketseat.

O modelo realiza a previsão do salário de um profissional com base em:

- `tempo_na_empresa`
- `nivel_na_empresa`

##  Etapas do Projeto

-  Análise Exploratória dos Dados (EDA)
-  Validação com K-Fold
-  Regressão Linear
-  Regressão Polinomial
-  Pipeline com Scikit-Learn
-  API com FastAPI
-  Interface em Streamlit

##  Tecnologias

- Python
- Pandas
- NumPy
- Scikit-Learn
- FastAPI
- Streamlit
- Joblib

##  O que Aprendi

Durante este projeto, aprendi a:

- Utilizar **K-Fold Cross Validation** para avaliar modelos de Machine Learning de forma mais confiável.
- Desenvolver e disponibilizar uma **API REST** utilizando **FastAPI** para servir as predições do modelo.
- Criar uma interface web simples com **Streamlit**, consumindo a API para realizar previsões.
